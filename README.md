# amandalynn.media

These are my notes for this website, which is my personal portfolio GitHub Pages 
site. These notes are mainly for myself, however, if you have found your way 
here I hope you find them helpful. 

[GitHub Pages](https://pages.github.com/) provides a completely free hosting 
environment for front-end content. You get a free subdomain, or you can 
[configure a custom domain](https://help.github.com/en/github/working-with-github-pages/about-custom-domains-and-github-pages).

This site has been built using the [Jekyll static site generator](https://jekyllrb.com/).
Jekyll provides a framework for building sites using Markdown, Liquid, HTML, 
and CSS.
- Jekyll uses [Markdown Layouts](https://jekyllrb.com/docs/step-by-step/04-layouts/) 
    to create pages rapidly using the same layout.
- [Liquid](https://github.com/Shopify/liquid/wiki/Liquid-for-Designers) is the 
    template engine, providing support for variables (Output) and logic (Tags).
